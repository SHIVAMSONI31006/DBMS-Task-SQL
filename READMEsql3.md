# Internship Task 3: Writing Basic SELECT Queries

## 📌 Project Overview
This project demonstrates how to extract, filter, sort, and limit data from a relational database using SQL. The queries are designed to run seamlessly via the terminal or command-line interface (CLI) using MySQL.

## 🛠️ Tools Used
* **Editor:** VS Code
* **Database Management System:** MySQL
* **Terminal Environment:** PowerShell / Command Prompt (CMD)

---

## 📁 File Structure
* `sqltask3.sql` - Main SQL script containing database setup, test data insertions, and evaluation queries.
* `README.md` - Documentation for the assignment.
